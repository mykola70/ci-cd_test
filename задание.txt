Задание
Разработка учебного проекта в котором нужно продемонстрировать: как работает CI/CD, изучить GitHub Actions, написать свой workflow-файл для автоматической сборки и деплоя Docker-контейнера на сервер, а также научиться использовать GitFlow и секреты (Secrets) для безопасного подключения.
Для этого:
1. Создай простой проект (например, Flask-или FastAPI-приложение).
2. Добавь в проект папку .github/workflows и создай в ней файл deploy.yml, который:
	запускается при push в ветку main,
	выполняет сборку Docker-образа,
	пушит образ в GitHub Container Registry.
3. Добавь job для деплоя на сервер через SSH-Action.
4. Проверь, что после коммита в ветку main workflow успешно отработал (см. вкладку Actions).
